Задача 5: Реализация LRU-кэша (Least Recently Used)
Уровень: Сложный
Задание:
Создай класс LRUCache, который реализует кэш с стратегией вытеснения "наименее недавно использованного" (LRU). Кэш должен:

Хранить элементы в виде ключ-значение.

Иметь ограниченный размер. При превышении размера удалять наименее использованный элемент.

Обеспечивать быстрый доступ к элементам (O(1) в среднем случае).

Требования:

Используй std::unordered_map для быстрого доступа по ключу.

Используй std::list для отслеживания порядка использования элементов (последний использованный элемент перемещается в конец списка).

Реализуй методы:

void put(int key, int value) — добавить/обновить элемент.

int get(int key) — получить значение по ключу (если ключа нет, вернуть -1).
